# MEDISECML - Secure Medical Image Encryption using ML

## Overview
This project implements a secure stream cipher framework for medical images using machine learning.

## Features
- ML-based key generation
- Image encryption & decryption
- Attack/anomaly detection
- Performance evaluation (PSNR, Entropy)

## Technologies Used
- Python
- NumPy, OpenCV, Matplotlib
- PyTorch

## Workflow
1. Load image
2. Generate key using ML
3. Encrypt image
4. Decrypt image
5. Detect anomalies
6. Evaluate metrics

## Results
- PSNR ≈ 100 (lossless recovery)
- Entropy ≈ 7–8 (high security)

## Run
```bash
python app.py
```
